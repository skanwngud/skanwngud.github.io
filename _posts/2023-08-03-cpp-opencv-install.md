## 설치환경
### MacOS
- MacBook Pro 14
- Apple M1 Pro
- Ventura 13.4.1

### Ubuntu
- Geforce 3090
- Ubuntu 20.04LTS

## 설치 (MacOS)
1. `brew install opencv` 를 통해 설치한다.
2.  `/opt/homebrew/Cellar/opencv/4.8.0_1/include/` 에 설치 된 것을 확인한다.
